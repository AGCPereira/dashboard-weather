# dashboard-weather

## Description
A website that allows you to input any city and views:
* current temp
* wind
* humidity
* UV index

Also gives a 5 day forecast of searched city and alows you to view the predicted:
* temperature 
* wind
* humidity

Website will also allow you to view search history
